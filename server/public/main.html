什么是ES6？

  • ECMAScript 6.0（以下简称 ES6）是 JavaScript 语言的下一代标准，已经在2015年6月正式发布了。它的目标，是使得 JavaScript 语言可以用来编写复杂的大型应用程序，成为企业级开发语言。

es6新语法

  • let和const
  • 这两个关键字与var相同,都是用来声明变量的.

1.es5里面只有全局作用域和函数作用域.没有块级作用域

                    var a = 3;
                    function fun(){
                      console.log(a);//
                      a = 4;
                    }
                    fun();

2.let在es6中增加了块级作用域.用它声明的变量,只在let命令所在的代码块内有效.

                    例子1:
                    var a = 3;
                    if(a == 3){
                      let b = 4;
                    }
                    console.log(b); // not defined
                    例子2:
                    let a = 3;
                    if(true){
                      let a = 4;
                      console.log(a); //4
                    }

3.有一个var带来的不合理场景.原因是变量泄漏为全局变量,覆盖了之前的声明.

                    var a = [];
                    for(var i = 0;i<10;i++){
                      a[i] = function{
                        console.log(i);
                      }
                    }
                    a[6](); // 10
                    a[0](); // 0
                    // 用let可以有效的解决此不合理场景.

4.const用来声明常量,常量一旦声明,值就不可以改变(引用数据类型的指针不可改变,但内容可以变更);

变量的解构赋值

  • es6允许你用一定的模式,从数组或者对象中提取值,对变量进行赋值,这就是解构(Destructuring).
  • 1.数组的解构

    •                       ->以前的数组操作
                        var arr = [1,2,3];
                        let a = arr[0];
                        let b = arr[1];
                        let c = arr[2];
                      ->es6允许写成如下格式
                        let [a,b,c] = [1,2,3];
                        // 解构可以从数组中提取值,按照对应位置,对变量进行赋值.

嵌套数组的解构

                  let [a,[b,[c]]] = [1,[2,[3]]];

                  let [,,c] =['白百何','文章','失恋33天'];
                  console.log(c);// '失恋33天'

                  var arr = [1,2,3,4,5,6,7];
                  let a = arr[0];
                  let b = arr[1];
                  let c = arr.slice(2);

                  let [a,b,...c] = [1,2,3,4,5,6,7];
                  console.log(c);// [3,4,5,6,7]

                  let [a,b,c] = [1,2];
                  console.log(c);//null undefined

                  // 如果解构不成功,变量的值就等于undefined

默认值

                  解构赋值允许指定的默认值.
                  let [a = 1] = []; 
                  console.log(a);// 1
                  let [x,y = '456'] = ['a'];
                  console.log(x);//a
                  console.log(y);//456

默认值可以引用解构赋值的其他变量,但该变量必须已经声明

                  let [x = 1,y = x] = [];
                  console.log(x);//a
                  console.log(y);
